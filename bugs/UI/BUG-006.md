# Bug Report

## Bug ID
BUG-006

## Description
Submitting an empty employee form with both add and edit modals does not show an error on the UI

---

## Test Environment

| Field          | Details            |
|----------------|--------------------|
| Browser        |        Chrome            |
| Browser Version|         147.0.7727.15           |
| Date Reported  |         03/05/2026          |

---

## Steps to Reproduce

1. Login to the system
2. Click the add employee button on the bottom of the grid
3. Click the 'Add' button on the modal without filling any of the fields

---

## Actual Result

No error appears on the UI, only an API error messasge on the Network tab

**Screenshot:**

![Actual Result](/bugs/screenshots/bug_006_actual.png)

---

## Expected Result

There should be an error shown on the UI to indicate the user that those fields are required to submit the modal
